Line Splitters or Dividing Rules

It is convenient to present pages divided by topics. The <hr>tag inserts a rule completely across the screen. The rule will break the page wherever inserted. It will appear black and scale width as the browser window.

Another way to divide documents into sections is by using graphics as rules. You make the graphic rule in a paint program and insert it with the <img>tag. Graphic rules do not change width when the browser screen is resized. The rule will not break the copy unless you use the <p> or <br> tags.
