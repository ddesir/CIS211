Difference Between HTML and CSS

Separation of Content from Design

HTML

HTML stands for Hyper-Text Markup Language. It is used to structure the content of a web page and can be considered the bones of the web page. 

					
<main>
	<h1>My Life</h1>

	<table>
		<tbody>
			<tr><!-- Row 1 -->
				<td><a href = '../../../ASSIGNMENTS/DEODES10.html'>DEODES10</a></td>
				<td><a href = 'professions.html'>CIS Professions</a></td>
				<td><a href = '../../../ASSIGNMENTS/DEODES30.html'>DEODES30</a></td>
			</tr>

CSS

CSS, or Cascading Style Sheets is used to style the structured markup of the HTML. If HTML is the bones, CSS is the flesh. 

					
table {
	table-layout: fixed;
	margin: auto;
	border-top: 1px solid #000;
	border-bottom: 1px solid #000;
	width: 100%;
}
