Difference Between HTML and CSS

HTML and CSS are the core languages for building webpages and web based applications. HTML stands for Hyper-Text Markup Language. It is used to structure the content of a web page and can be considered the bones of the web page. CSS, or Cascading Style Sheets is used to style the structured markup of the HTML. If HTML is the bones, CSS is the flesh.

HTML CSS
Definition HTML is a markup language for describing web documents (web pages). CSS describes how HTML elements are to be displayed on screen, paper, or in other media.
Basics Dictates content and structure of a web page. Modifies design and display of HTML elements.
File Differences You can use CSS in an HTML file. You can not use HTML in a CSS style sheet.
Uses Publishes Documents online Design colors, fonts and layout
Retrieve online info Adapt display across platforms
Design forms Easier site maintenance
Include apps in their documents Tailor multiple pages using a single stylesheet
Examples HTML consists of tags surrounding content
<tag>content</tag>		 CSS consists of selectors followed by a declaration block
{property: value;}
