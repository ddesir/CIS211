Definitions

Network: A telecommunications network which allows computers to exchange data and information.

Host: Main computer and distributor of information

Server: A powerful computer able to share and use

Client - Server Network: Commonly used topology of connectivity between main computers and workstations

Local Area Network: (LAN) A config of several computers connected together located in the same place

WAN: (Wide Area Network) Computers located in different places in a distributed environment sharing data and information remotely

The Difference Between Hypertext and Hyperlink

Hypertext is an expression, bold or underlined, that indicates by clicking a new page will show up on the screen. Hyperlink, the name of a page or a site that will come up once you click on the hypertext.

Web Server: A computer that serves up web pages when requested by a client

Web Browser: A program that used for displaying and viewing pages on the world wide web

Web Page: A hypertext document connected to the world wide web
