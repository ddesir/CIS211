Input devices are used to get information into the computer. Devices such as the keyboard, mouse and camera can be considered input devices. Output devices are used to get information out of the computer, they can include monitors or displays and printers.

Memory on a computer usually refers to RAM, or Random Access Memory, and is where data is loaded temporarily while a process runs. Storage is where data is stored for long term retention, as in on a hard drive.

Non-volatile memory is used for secondary or long-term storage and volatile storage is used for primary storage. Volatile storage only lasts for as long as the computer session is going. An example of volatile storage is RAM and an example of non-volatile storage is a hard drive.
