This Semester

This semester, I learned HTML5, CSS3, and a little Javascript. While everything I learned, may not have been learned as part of the course material, I learned a lot. I learned enough to be confident enough to experiment while building out this site.

Code and Languages

HTML5

During the course of CIS 211, I learned basic HTML5 syntax and best practices. I learned how to build documents, validate my code and to use more semantically correct HTML.

CSS3

This semester, I also learned how to style my HTML with CSS. I used pseudo-classes, selectors, and CSS animations. While, at the time I'm tapping this out, I used flex box, I didn't not fully learn the concept. I am in a good place to take my knowledge further.

Javascript

For the first time, I explored some of the HTML Document Object Model with Javascript. I wanted to do some things that weren't possible with HTML and CSS alone, but I knew that some scripting could make those things happen. I wrote the date / time function before it was provided and tweaked it a bit afterwards.

All My Own

To build this site, I used HTML, CSS, and Javascript. Every "bit" was not crafted from scratch by me, but I did learn from what was there and I took it and made it all my own. Prior to this site, I may have seen CSS animations, and didn't know what they were. I have attempted to learn Javascript, but I've never learned or used it in the context of web development.

I set out to learn as much as possible and make everything I present in these pages my own. I used Python to generate maps, so that it would match my color scheme and highlight areas that I wanted, as in the case of the markers on the world map or the rivers on the rivers page.
